## ✨ Features

This project consists of two seamlessly integrated pages featuring modern web design trends:

- **Bento Grid Layout (Home):** A clean, modular dashboard-style layout for the main profile, links, and contact information.
- **Masonry Layout (Gallery):** A fluid, Pinterest-style grid for showcasing artworks without awkward gaps or cropping.
- **Synchronized Dark/Light Mode:** A custom theme toggle that saves user preference in `localStorage` and smoothly persists across both pages.
- **Interactive Lightbox:** A custom-built Vanilla JS image modal with smooth scale-in animations for viewing full-resolution artworks.
- **Glassmorphism UI:** Subtle transparent backgrounds, glowing gradients, and smooth 3D tilt hover effects.
- **Fully Responsive:** Adapts perfectly to desktop, tablet, and mobile screens.
